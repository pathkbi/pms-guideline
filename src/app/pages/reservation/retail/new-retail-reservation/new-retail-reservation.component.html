How to create new Reservation

Step 1: Navigate to "Create Booking" Screen
Step 2: Input Booking infomation
  • Arrive time Pick the date guest want to check-in
  • Departure time Pick the date guest want to check-out
  • Room type Choose a room type of list
  • Adults Input a number of adults
  • Children Input a number of children
  • Accommodation service there are 2 type of Accommodation: Long term and short term
  • Guest types
  • Agency
  • Rate code Depend on the chosen rate code, the price will show
  • Payment method
  • Room discount by rank there are 2 types of discount: pecent or amount
  • Status there are 3 statuses: definite, tentative, waiting
  • Release time If the status is "Tentative", you should choose the time to release that room
Step 3: Input Guest infomation
  • Full name
  • Phone number
  • Email
  • Gender
  • ID Card
  • Passport
  • Passport expried date
  • Nationality
Functions:

Select Room Assign specific room to this reservation

Deposit Open the deposit popup

Open price Allow adjust the price directly

Additional services open the popup to allow add more services.

Upload image open the popup to upload customer document images.

Save reservation and Print all input datas will be save and open the popup to print "REGISTRATION CARD"

Save reservation all input datas will be save
